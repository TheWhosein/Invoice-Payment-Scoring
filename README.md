# Invoice-Payment-Scoring

**Goal:** To predict the likelihood of invoice payment defaults using Classic ML algorithms, helping financial institutions identify high-risk clients early.

**Technologies Used:**
* **Python:** Core programming.
* **Pandas & NumPy:** Data cleaning and preprocessing.
* **Scikit-learn:** Model training (`RandomForestClassifier`).

**Result:**
* Achieved **~85% accuracy** in predicting high-risk invoices, streamlining the credit approval process.

**How to Run:**
Simply open the `Invoice-Payment-Scoring.ipynb` file and run the cells to see the data processing and final results.
